Our Projects Experience

Equax team has worked on various E-Governance and specific customer oriented applications as projects, who studied, understood the end-user methodologies and processes to make the application user acceptable. Some of these applications are:
  • Insurance Regulatory Development Authorities – Agent License Portal (2002)

    License maintenance portal for generation of New & Renewal agent licenses by all the Insurance Companies in India. A dedicated team supported IRDA portal which was developed, maintained and hosted by MIT until 2001 - 2009 Oct. This was developed in ASP with COM+ VB components and MS SQL 2000 on Microsoft Server 2000 with both Microsoft Transaction Server & Internet Information Server.

  • Road Transport Office (RTO) – e-RTA for Vehicle Registration Portal (2003)

    E-RTA portal for both Learning & Permanent Driving Licenses for GOA – RTA which has for the first time in India - KIOSK’s introduced for issue of learning licenses. This was developed & maintained and hosted at GOA-NET along with Goa Electronics Ltd., from 2002 onwards. This was also developed in ASP with COM+ VB components and MS SQL 2000 on Microsoft Server 2000 with both Microsoft Transaction Server & Internet Information Server.

  • Road Transport Office (RTO) – e-RTA for Vehicle Registration Portal (2003)

    E-RTA portal for Registration of Vehicles for GOA – RTA which has for the first time in India had dealer vehicle stock available for RTO’s for approval and approved stock were registered by taking the vehicle details like CHASSIC#, COLOR, MODEL, Etc., This was developed & maintained and hosted at GOA-NET along with Goa Electronics Ltd., from 2003 onwards. This was also developed in ASP with COM+ VB components and MS SQL 2000 on Microsoft Server 2000 with both Microsoft Transaction Server & Internet Information Server.
